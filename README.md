# FileManager
简单的文件管理器，打开SD卡目录，用列表显示文件夹及其文件，文件夹显示下面的文件数，文件显示大小。


图片，Word，PPT，excel,PDF文件的打开调用系统其他应用，TXT文件直接打开。
